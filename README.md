SupportLink – Live Chat Support System
SupportLink is a real-time customer support chat system built using the MERN stack.
It includes:
	•	A customer-facing electronics website with an embedded chat widget
	•	A separate agent dashboard where support staff handle customer queries
The system uses Socket.IO for real-time messaging, JWT authentication for agents, and MongoDB for chat persistence.
This project demonstrates full-stack development, real-time communication, and role-based access control in a practical customer support scenario.
Tech Stack: React, Node.js, Express, MongoDB, Socket.IO, JWT
